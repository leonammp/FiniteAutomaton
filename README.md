# FiniteAutomaton
Deterministic Finite Automaton Simulator

## Configuration
Set the automaton struture in file: <b>automaton.txt</b>
### * states
In this variable must have the states along with their transitions.<br>
Exemple <br>
<pre>
S0 : { 'a' : 'S1', 'b' : S2 }
</pre>

### * start
The initial state of the automaton
 
### * ends
Set of final states

Code in Python 2.7
